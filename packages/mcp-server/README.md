# @iris/mcp

The Iris Model Context Protocol server. Exposes `find_skill` / `load_skill` / `run_skill_script`, `skill://` resources and `iris:` prompts so any MCP client (Claude Code, Codex, Cursor, Gemini CLI) can discover and invoke skills. Binary: `iris-mcp`.

Part of [**Iris**](https://github.com/BryanGabrielAtangana/iris) — an open protocol + reference implementation that lets any agent reliably discover and invoke the right skill at the right time, from a portable skill library.

> Iris knows every capability you own and hands the right one to whatever agent you're working with, exactly when it's needed.

See the [root README](https://github.com/BryanGabrielAtangana/iris#readme) for the full picture and the [protocol spec](https://github.com/BryanGabrielAtangana/iris/blob/main/spec/skill-access-protocol.md).
