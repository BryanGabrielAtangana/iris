# `@iris/docs` (deferred)

> **Status: deferred — a later milestone.** This directory is a placeholder for
> the Iris documentation and specification site. There is no site to build or run
> here yet.

Iris knows every capability you own and hands the right one to whatever agent
you're working with, exactly when it's needed. This app will eventually host the
public docs and a rendered version of the protocol spec so that story is easy to
read, search, and link to.

## Planned scope

- A docs site covering concepts, the CLI, the MCP server, and the adapters.
- A rendered, browsable version of the skill-access protocol specification.
- Guides and examples, including the [quickstart](../../examples/quickstart/README.md).
